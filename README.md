int2word
========
[![GoDoc](https://godoc.org/github.com/mohae/int2word?status.svg)](https://godoc.org/github.com/mohae/int2word)[![Build Status](https://travis-ci.org/mohae/int2word.png)](https://travis-ci.org/mohae/int2word)

int2word takes an `int64` and creates its word equivalent in English using American names for large numbers, e.g. 10^12 is a trillion.
